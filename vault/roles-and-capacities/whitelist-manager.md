# Whitelist Manager

The Whitelist Manager is responsible for controlling which addresses are authorized to interact with the vault.

{% hint style="info" %}
This role is only applicable if the whitelist feature was enabled during the vault's creation.
{% endhint %}

#### Capabilities: <a href="#capabilities" id="capabilities"></a>

* **Add/Remove Whitelisted Addresses**: Manages the whitelist by adding or removing addresses, ensuring only authorized users can access the vault.

This role is vital for controlling access, ensuring only designated addresses interact with the vault while balancing flexibility and security.

{% hint style="info" %}
Only the admin can disable the whitelist
{% endhint %}
